A Quiz app that lets users login and signup, and has a universal leaderboard for scores and a profile that lets you see your history and email

--Main Features--

Home Page — entry point for the quiz

Quiz Page — dynamic 10-question quiz from API

Results Page — displays the user’s score after completion

Signup & Login System — JWT authentication + password hashing

Score Saving — stores quiz results in MongoDB

Profile Page — shows user info & full quiz history

Leaderboard — top 10 players ranked by highest score

--Extra Features Implemented--

User can choose number of questions

User can choose difficulty (Easy / Medium / Hard)

Ability to restart/replay the quiz

Fully deployed on Render

HTML decoding + answer randomization

Clean UI with shared navigation bar

--Frontend--

HTML5

CSS3

JavaScript (Vanilla JS)

--Backend--

Express.js
